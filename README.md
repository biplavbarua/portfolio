# 🚀 Biplav Barua — Portfolio

A modern, responsive developer portfolio built with **React 18**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**. Features a clean dark/light theme toggle, smooth scroll animations, and a fully responsive layout.

![TypeScript](https://img.shields.io/badge/TypeScript-5-3178C6?logo=typescript&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-3-06B6D4?logo=tailwindcss&logoColor=white)
![Vite](https://img.shields.io/badge/Vite-5-646CFF?logo=vite&logoColor=white)

## ✨ Features

- **Dark / Light Mode** — System-aware theme toggle with smooth transitions
- **Animated Sections** — Scroll-triggered animations powered by Framer Motion
- **Responsive Design** — Mobile-first layout that works on all screen sizes
- **Open Source Impact** — Showcases merged PRs to projects like Pandas, Monkeytype, Mole, and GlobalSpeed
- **Project Filtering** — Filter projects by category (Data Science / Web)
- **Typewriter Effect** — Rotating role titles in the hero section
- **SEO Optimized** — Semantic HTML, proper heading hierarchy, and meta tags

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 18 + TypeScript |
| Styling | Tailwind CSS 3 |
| Animations | Framer Motion |
| Build Tool | Vite 5 |
| Icons | Lucide React |
| Deployment | Vercel / Netlify |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/biplavbarua/portfolio-v2.git
cd portfolio-v2

# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
