## ⚡ What is this?

My personal developer portfolio — not a template, not a clone. Designed from scratch with obsessive attention to detail, smooth animations, and real content.

```
⏱  Built in a weekend    🎨  Dark + Light themes    📱  Fully responsive    🚀  Blazing fast
```

## 🏗️ The Stack

```
 React 18 ──── Component architecture & hooks
    │
 TypeScript ── Type safety across the entire codebase
    │
 Tailwind ──── Utility-first responsive styling
    │
 Framer ────── Scroll-triggered animations & transitions
    │
 Vite ──────── Sub-second HMR & optimized builds
```

## 🚀 Run It Locally

```bash
git clone https://github.com/biplavbarua/portfolio.git
cd portfolio
npm install
npm run dev        # → http://localhost:5173
```

```bash
npm run build      # Production bundle
npm run preview    # Preview production build
```

## 📁 Architecture

```
src/
├── components/
│   ├── Hero.tsx            # Typewriter + animated avatar
│   ├── Experience.tsx      # Timeline with skill tags
│   ├── Education.tsx       # Card layout with highlights
│   ├── Projects.tsx        # Filterable gallery
│   ├── Skills.tsx          # Category grid + certs
│   ├── Achievements.tsx    # OSS merged PRs showcase
│   ├── Navbar.tsx          # Scroll spy + mobile menu
│   ├── ThemeToggle.tsx     # Dark ↔ Light switch
│   └── Footer.tsx          # Socials
├── utils/cn.ts             # clsx + tailwind-merge
├── App.tsx                 # Section orchestrator
├── index.css               # Design tokens + themes
└── main.tsx                # Entry point
```
